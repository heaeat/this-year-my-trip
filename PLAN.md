# 기획 / 논의 중인 항목

## 교통편 탭 확장

### 배경
- 현재 `항공편` 탭은 비행기 전용
- 기차(KTX/SRT), 고속버스, 배(페리)는 교통 카테고리(🚌)로 묶여있음
- 이들은 **출발지/도착지/시간 정보가 중요**하지만 가계부 카드로만 표시되어 스케줄 파악이 어려움
- 택시는 간단히 금액만 기록하면 됨

### 검토한 방향
1. **교통 카테고리에 서브타입 추가** (❌ 포기)
   - 다른 카테고리는 카테고리 선택 → 바로 상세 필드
   - 교통만 서브타입 선택 단계가 추가되면 UI 흐름이 깨짐

2. **교통 카테고리를 여러 개로 분리** (🚄 기차 / 🚌 고속버스 / ⛴️ 배 / 🚕 택시) (❌ 포기)
   - 카테고리가 9→12개로 늘어남
   - 카테고리 요약/필터 UI가 복잡해짐

3. **항공편 탭을 "교통편" 탭으로 확장** (✅ 유력)
   - 기존 항공편 탭의 스케줄 표 레이아웃 재활용
   - 항공 + 교통(택시 제외) 항목을 함께 표시
   - 타입 구분은 아이콘으로 (✈️ / 🚄 / 🚌 / ⛴️)

### 구현 시 결정 필요한 것
- **교통 유형 입력 방법**
  - (a) 예약 모달에 카테고리 "교통" 선택 시 유형 선택 드롭다운 (🚄/🚌/⛴️/🚕)
  - (b) 이름에서 키워드 자동 감지 (KTX, 버스, 배 등) — 부정확할 수 있음
- **데이터 모델**
  - `r.transportDetails = { type: 'train'|'bus'|'ferry'|'taxi', from, to, fromTime, toTime, fromDate, toDate, vendor, bookingNo }`
- **탭 라벨**
  - 🚀 교통편 / ✈️ 교통편 / 🧭 교통편 중 고민
- **택시 처리**
  - 교통편 탭에선 제외 (스케줄 없음)
  - 가계부 탭에서만 표시
